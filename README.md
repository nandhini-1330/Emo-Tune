Emo-Tune

Emo-Tune is a smart music recommendation system that detects a user's emotional state and recommends songs accordingly. The system supports three independent modes of emotion input — Text, Voice, and Face Detection (Camera) — in addition to a Quiz-based mode for mood assessment.

Overview

Traditional music players require users to manually search or select playlists. Emo-Tune removes this friction by analyzing the user's current emotional state and automatically curating song recommendations that match their mood, using whichever input method the user prefers.

Features


Multiple Emotion Input Modes

Text Input — Users describe their mood in text, which is analyzed to determine emotion.
Voice Input — Speech is captured and analyzed to detect the user's emotional tone.
Face Detection (Camera) — Real-time facial expression analysis via webcam to identify emotion.


Quiz Mode — A short, structured questionnaire as an alternative method to determine mood.
Emotion-Based Recommendations — Songs are recommended and played based on the detected emotion category (Happy, Sad, Angry, Neutral).
Language Selection — Users can choose a preferred language before proceeding.
Lightweight Front-End Architecture — Built entirely with HTML, CSS, and JavaScript; no backend server required to run.
